# TIL [JSP 태그, JSTL]

## JSP
* Java 코드를 페이지 내에서 작성할 수 있도록 함.

### 스트립트릿 태그
* Java 코드를 JSP 페이지 내에 작성할 때 사용.
* e.g.) 변수 선언, 반복문 등 Java 로직 처리
~~~
<% 
    String name = "Hyun";
    int age = 30;
%>
~~~

### 선언문 태그
* JSP 페이지 내에서 메서드나 변수 선언을 할 때 사용.
* JSP 파일의 스코프 내에서 사용할 수 잇는 메서드와 변수 선언.
~~~
<%!
    public int add (int a, int b)
    {
        return a+b;
    }
%>
~~~

### 표현식 태그
* Java 코드를 실행하여 결과를 화면에 출력하는 데 사용.
* 출력값을 바로 HTML로 전환하여 화면에 표시.
~~~
<%=
    <p>이름 : <%=name></p>
    <p>나이 : <%=age></p>
    <p>더하기 : <%=add(1,2)></p>
%>
~~~

## JSTL
* JSTL은 JSP 페이지에서 자주 사용하는 기능을 태그 형태로 제공하는 라이브러리.
    * 조건문 : `<c:if>`
    * 반복문 : `<c:forEach>`
    * URL 처리 : `<c:url>`
