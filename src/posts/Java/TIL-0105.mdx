# TIL [Java 자료형과 배열]

## 자료형
* 변수에 저장될 데이터의 형태 (스택 메모리에 저장)

### 기본 자료형 (byte 크기)
* 정수형
    * byte, short, int, long (1, 2, 4, 8)
* 실수형
    * float, double (4, 8)
* 문자형
    * char (2)
* 논리형 
    * boolean (1)
### 기본 자료형의 특징
* 값 자체를 저장, 크기 및 범위 고정
* 자료형에 따라 초기화된 값이 존재
    * int : 0
    * boolean: false
    * char : \u0000 등

### 참조 자료형
* 객체, 배열 같은 복잡한 데이터 구조를 다룰 때 사용.
* 변수에 데이터 메모리 주소를 저장, 주소를 통해 실제 데이터를 참조. (힙 메모리에 저장)
* 배열: int[], String[]
* 객체: 클래스타입 (String, Class 등)

### 참조 자료형의 기본 값
* 배열 자체 : <code>null</code>
* 배열 요소 : 데이터 타입에 따라 초기 값이 달라짐.
* 예시 : <code>String</code>은 참조 자료형이므로, 기본 값은 <code>null</code>


~~~
// 기본 자료형
int x = 10;  // x 변수 값 10이 저장

// 참조 자료형
String str1 = "Hello";  // str1은 "Hello" 문자열이 저장된 메모리 주소를 가리킴.
MyClass obj1 = new MyClass();  // obj1은 MyClass 객체가 저장된 메모리 주소를 가리킴. (힙 메모리에 저장)
obj1.name = "John"; // obj1이 가리키는 객체의 속성에 접근.
~~~

## 배열의 정의
* 나열된 데이터의 집합
* 일정한 크기의 연속적인 메모리 공간을 사용, 여러 값을 하나의 변수로 관리.
* 배열은 객체로 취급

## 배열의 특징
* 배열은 생성 시 크기가 고정, 크기 변경 불가
* 배열 요소는 0부터 시작하는 인덱스를 사용해 접근.

~~~
// 정수형 배열 생성
int[] arraybox = new int[9];

System.out.println(arraybox[0]); // 출력: 0

String[] strArray = new String[5];
System.out.println(strArray[0]); // 출력: null
~~~
